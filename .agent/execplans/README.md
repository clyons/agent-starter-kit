# ExecPlans Directory

This directory stores living execution plans for substantial work.

## Naming Convention

Create one file per initiative:

- `.agent/execplans/YYYY-MM-DD-<short-kebab-title>.md`

Example:

- `.agent/execplans/2026-02-13-auth-token-hardening.md`

## How To Start

1. Copy `.agent/execplans/TEMPLATE.md` into a new dated file.
2. Fill in all sections before implementation.
3. Keep the file current while implementing.
4. Link the plan path in the PR description.

## Required Companion Files

- `AGENTS.md`
- `.agent/PLANS.md`
